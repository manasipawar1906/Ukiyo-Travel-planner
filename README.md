# Ukiyo

Ukiyo is an AI-powered travel itinerary generator that creates personalized travel plans using Groq AI.

## Features

- AI-generated travel itineraries
- Destination-based recommendations
- Fast itinerary generation using Groq API
- Responsive user interface

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python Flask

### AI
- Groq API

## Project Structure

ukiyo/
├── assets/
├── backend/
└── frontend/

## Run Locally

pip install -r backend/requirements.txt

python backend/app.py
